**Interview-AI Platform**


Interview-AI is an intelligent, interactive mock interview platform designed to simulate real-world technical interviews. Built using React and Tailwind CSS, it provides a sleek, responsive UI and leverages AI (via OpenAI) to generate dynamic, domain-specific interview questions and evaluate answers in real-time.
